# Plano de Finalização do Projeto StockPro

## Visão Geral do Projeto
O StockPro é um aplicativo de controle de estoque mobile-first (React Native/Expo) com Firebase (Firestore + Auth). O projeto já tem a maior parte das funcionalidades implementadas.

---

## Fase 1: Correções Críticas (Bloqueantes)

### 1.1 Login e Autenticação
- **Problema**: Usuário diz que login "entra direto" 
- **Análise**: O Firebase Auth persiste sessão por padrão - comportamento esperado
- **Solução**: Já está correto. Usuário precisa fazer logout para ver tela de login
- **Ação**: Nenhuma - comportamento correto do Firebase

### 1.2 Drawer/Navegação
- **Problema**: Botões do drawer não respondem
- **Análise**: O drawer está implementado com Animated. Pode haver conflito com a animação.
- **Ação**: Revisar código do drawer em `app/(tabs)/_layout.tsx`

### 1.3 Teste de Integração Firebase
- Verificar se todas as operações (CRUD) funcionam corretamente
- Garantir que userId está sendo enviado em todas as operações
